计算机的基本硬件组成

计算机最基本的三大件就是，CPU、内存、主板

CPU

        在计算机的三大件中，CPU是最核心的配件，全名叫做中央处理器(Central Processing Unit)。为什么说CPU是“最重要”的呢，因为计算机所有的计算都是由CPU来计算。所以CPU也成了计算机中造价最贵的部分之一

内存(Memory)

​ 第二个重要的配件就是内存，你在计算机中所有运行的程序，比如打开浏览器，运行游戏，都要加载到内存中才能运行。程序读取的数据，计算得到的结果也要反倒内存里，内存越大，加载的东西就越多。这也是我们平时买电脑时喜欢买大内存的，因为可以同时跑很多的程序。

主板

​ 主板其实就是一个有着各种各样的插槽的配件。存放在内存里的程序和数据需要CPU读取，CPU计算完之后需要把数据写回内存，然后CPU没有直接插在内存上，所以就需要一个沟通桥梁，这个桥梁就是主板。


我们的CPU和内存条都需要插在主板上，主板的芯片组(Chipset) 和总线(Bus)解决了CPU和内存之间如何通信的问题。芯片组控制了数据传输的流转也就是数据从哪来到哪去的问题。而总线就是数据传输的高速公路，所以总线速度决定了数据传输的有多快

=================================================================================

​ 计算机只要有了这三大件，然后在配上电源，差不多就可以跑起来了。但是这个东西我们人类现在还没法使用它，因为它还缺少输入/输出设备 也就是I/O设备。这样我们才让计算帮助我们干活并且得到反馈。像我们个人电脑必须得有个显示器，因为只有它我们才能看到各种图像，文字。显示器这种设备也就是输出设备。像鼠标，键盘这些配件就是输入设备。最后再给电脑配置一个硬盘，这样各种数据才能持久的保存下来。

​ 现在电脑上基本上都会配备一个风扇用于解决灰尘和散热问题，不过这不是计算机必备的硬件。除此之外，计算机还有个特殊的设备，这个设备就是显卡(Graphice Card)。现在的图形界面操作系统的计算机，显卡都是必不可少的。可能有人也会有疑问，我买的电脑的时没有显卡也可以看见各种图形啊，那是因为现在的主板都带了内置的显卡。如果想要玩大型的游戏，你可能还需要单独的显卡(简称独显)插在主板上，显卡之所以特殊，是因为显卡里面除了cpu之外还有另外一个“处理器” ，也就是GPU(Graphics Processing Unit，图形处理器)。GPU同样可以做各种计算工作

​ 鼠标、键盘这些外部I/O设备，它们是通过主板上的南桥芯片组来控制和CPU通信的。
